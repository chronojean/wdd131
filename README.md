# WDD 131 - Dynamic Web Fundamentals

This repository contains my coursework and projects for the **WDD 131: Dynamic Web Fundamentals** course at Brigham Young University-Idaho.

## Author
**Jean Andrade**

## Repository Structure
- `index.html`: Course portal home page.
- `styles/`: Contains all CSS stylesheets for the projects.
- `scripts/`: Contains all JavaScript files.
- `images/`: Stores all optimized images used in the assignments.

## Course Sprints
- **Week 1**: Setup & Introduction (Home Page)
- **Week 2**: Responsive Design & DOM (Picture Album)
- **Week 3**: Responsive Images & Functions (Country Page)
- **Week 4**: Tables, Lazyloading, JS Objects (Picture Album Enhancement)
- **Week 5**: Forms & localStorage (Product Review Form)
- **Week 6**: Final Project
- **Week 7**: Final Exam

## Technologies
- HTML5
- CSS3 (Vanilla, Responsive Design)
- JavaScript (Vanilla, DOM Manipulation)
